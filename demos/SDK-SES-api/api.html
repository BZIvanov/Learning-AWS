General info

Request and response body is always of type JSON.

Quick access

createTemplate | deleteTemplate | getAccountSendingEnabled | getSendStatistics | sendEmail | sendTemplatedEmail | verifyEmailIdentity

Verify email address

Request

Key Value
Endpoint http://localhost:3100/verifyEmailIdentity
Method POST
Example body 
{
  "mail": "biser.ivanov@gmail.com"
}

Response

Key Value
Example body 
{
  "ResponseMetadata": {
    "RequestId": "bb0d47f7-2056-4168-9d75-ebc519316506"
  }
}
