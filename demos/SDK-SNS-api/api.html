General info

Request and response body is always of type JSON.

Quick access

createPlatformApplication | createPlatformEndpoint | createTopic | getEndpointAttributes | getTopicAttributes | listEndpointsByPlatformApplication | listPlatformApplications | listSubscriptions | listTopics | subscribe

Create topic

Request

Key Value
Endpoint http://localhost:3100/createTopic
Method POST
Example body 
{
  "name": "MyTestTopic"
}

Response

Key Value
Example body 
{
  "ResponseMetadata": {
    "RequestId": "d58a5104-6ff1-5a8b-a796-0983081760f5"
  },
  "TopicArn": "arn:aws:sns:us-east-1:134956957592:MyTestTopic"
}
