Policies

About

Policies could be attached to users and groups. If attached to a group it will be applied to all the users within the group.

Policy example

Sid stands for statement id

{
   "Version": "2012-10-17",
   "Statement": [
      {
         "Sid": "FullAccess",
         "Effect": "Allow",
         "Action": ["s3:*"],
         "Resource": ["*"]
      },
   ]
}
