Basic usage with MySQL

  1. From the search bar, search for RDS

  2. Click Create Database. Then proceed with Standard create and choose MySQL.

  3. After few minutes your db will be ready.

  4. Open your database to get the configurations for it.

  5. Test it with MySQL Workbench by making new connection and connecting to our new db.

  6. And now we can run some test queries to test our db
