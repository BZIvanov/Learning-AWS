Basic usage

  1. You have two main options: Manage User Pools (for working with users) and Manage Identity Pools (for working for example with google, facebook)

  2. Select the option Manage User Pools click the button Create a user pool. Now give it a name and select Step through settings

    This pool will manage users signed up in our application

  3. Proceed with selection as on the screenshots

    On the next step click Add an app client. Important here is to uncheck Generate client secret.

  4. Click Create Pool after reviewing the selected settings

  5. Set domain name for your app
