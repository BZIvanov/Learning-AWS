ECR - Elastic Container Registry

ECR is private Docker image repository. ECR allows us to store our docker images. It is similar to what Docker Hub does, but ECR is managed by Amazon.

Access to ECR is controlled through IAM.

Login command

This is the CLI v2 login command.

aws ecr get-login-password --region us-east-1 | docker login --username AWS --password-stdin 12345.dkr.ecr.us-east-1.amazonaws.com
