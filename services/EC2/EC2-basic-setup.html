EC2 - Elastic Compute Cloud

Basic usage

  1. From the search bar, search for EC2

  2. Find Launch instance button in the Dashboard or from the menu on the left click Instances and again click Instances

  3. From the Quick Start we will select with Linux 2 AMI 64-bit and click Select button. AMI = Amazon machine image

  4. Select the instance type and click Next: Configure Instance Details

  5. Select 1 instance and leave rest as default and scroll down

  6. Defaults here as well

  7. Copy the below commands in the optional textarea. These are commands which will run everytime we start our instance. After copy pasted the commands in the textarea click Next: Add Storage

    yum update -y
    yum install -y httpd
    systemctl start httpd
    systemctl enable httpd
    echo "Hello World from $(hostname -f)" > /var/www/html/index.html

  8. Click Next: Add Tags

  9. Click Add Tag button if you want to add tags. Tags are key:value pairs. After added all the tags you want click Next: Configure Security Group

  10. Here by default we only have the SSH row for our new security group. Click the Add Rule button and add HTTP type so we can access our instance from anywhere. Click Review and Launch

  11. On the next step review the selections and if everything is correct click Launch. Then from the dropdown select Create a new key pair which we will need later to access our instance with ssh from the local computer. Give it a name by your choice and click Download Key Pair. A .perm file will be downloaded which you don't have to lose, because you can't recreate it. Finally click Launch Instance

  12. Now from the Instances from the menu on the left you can see the list with your instances.

    The current state is Running, but we can also stop or terminate the instance from the button Instance state

    The public ip address is how we can access the instance from the browser. Note that everytime restart the instance public address will be different. The private address will always be the same.

  13. Now we can open the instance in the browser. Note that it should be http not https.
