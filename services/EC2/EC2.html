EC2 - Elastic Compute Cloud

OS options

We can choose between all Linux, Windows and MacOS. The most common choice is Linux.

Security groups

SGs control how the traffic is allowed into or out of our EC2 instances. They can reference by IP or by another Security group.

One SG can be attached to multiple EC2 instances.

If our application is not accessible (timeout) it means it is most likely security group issue. And if the connection is refused it is probably application error or it is not launched.

Common ports to know

  • 21 - FTP(File transfer protocol)
  • 22 - SSH
  • 22 - SFTP(Secure file transfer protocol), upload files using SSH
  • 80 - HTTP
  • 443 - HTTPS
  • 3389 - RDP(Remote desktop protocol), log into a windows instance
