Launch Types

  • On Demand - pay for what you use

    • billing per second for Linux and per hour for other OS
    • highest cost, but no upfront payment
    • no long-term commitment

  • Reserved

    Reserved instance types

    • Reserved - reservation period 1 or 3 years. Choose between payments: no upfront, or partial upfront or all upfront
    • Convertible Reserved - allow instance type change, for example m5.large to something else
    • Scheduled Reserved - for example every Monday

  • Spot - cheap, but unpredictable

    • you can lose the instance if you pay below the spot price which varies over time
    • if the spot instance is terminated by Amazon EC2, we will not be charged for a partial hour of usage. However if we terminate our instance we will be charged for the complete hour in which the instance ran.

  • Dedicated Host - physical server just for you

    • 3 years reservation period

  • Dedicated Instances

    • running on a hardware dedicated to you
