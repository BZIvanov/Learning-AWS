CloudFront

CloudFront is a CDN (Content Delivery Network)

It provides improved read performance by caching the content in the edge locations. It also provides DDoS protection.

Origins

S3

Distributing files and caching them at the edges.

Custom origin (HTTP)

CloudFront could target any of the following:

  • EC2 instance
  • Application Load Balancer
  • S3 Website
  • Any HTTP backend
